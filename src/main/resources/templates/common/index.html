RBAC系统

功能介绍：

本项目分为四大模块。

  1. 个人中心

    • 个人资料
      可以查看个人资料，包括用户名，邮箱，电话等。
    • 资料修改
      可以对自己的资料进行修改，用户名，邮箱，电话等
    • 密码修改
      可以对自己的密码进行修改
    • 我的权限
      可以看到我能够访问的地址

  2. 用户组管理

    • 用户组列表
      可以看到所有的用户组，包含删除，新增用户组
      删除用户组需要确保这个用户组没有任何用户
    • 用户列表
      可以看到当前的所有的用户，以及他们的角色，留言数量

  3. 权限管理

    • 角色列表
      角色列表，可以看到当前所有角色，可以添加角色，删除角色。
      删除角色之前需要确保这个角色没有被任何用户使用
    • 资源列表
      可以看档系统拥有的所有资源，能对资源进行删除，新增。
    • 角色分配
      可以看到所有的用户，以及他们的角色信息，留言数量。
      通过分配角色按钮进入新页面，可以给用户分配角色
    • 权限分配
      可以看到所有角色，通过分配权限按钮进入新页面，可以给角色分配权限

  4. 消息管理

    • 消息查看
      可以查看所有的消息，包含修改和删除功能。
    • 新增消息
      可以添加新的留言消息
